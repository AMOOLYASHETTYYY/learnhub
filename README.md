LearnHub – Online Learning Platform
Pace Wisdom Internship Project – 2025

A full-stack Django-based platform where teachers can upload courses with pricing, and students can explore, purchase, and learn.

Features

Student & Teacher Registration (with 2FA Email OTP)  
Login system with OTP verification  
Teachers can:
- Create, edit, and delete courses
- Add course content
- View enrolled students  
Students can:
- Explore and search for courses
- Subscribe/purchase courses
- Access subscribed content  
PayPal Payment Integration  
Email Notifications for:
- New content added
- New courses by subscribed instructors
